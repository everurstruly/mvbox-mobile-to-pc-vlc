# MyMovieBox

MovieBox Sync is a desktop tool for transferring and organizing media from your phone into a structure that works cleanly with VLC.

It scans connected mobile storage, identifies video content and related files such as subtitles, and restructures them into a more usable library on your PC.

## Overview

Media stored on mobile devices is often scattered across poorly named folders with inconsistent structure, especially when sourced from apps like MovieBox. This makes it difficult to move content to a desktop environment and use it with players like VLC without manual sorting.

MovieBox Sync automates that process by detecting likely media directories, evaluating their relevance, and organizing the results into a cleaner format.

## What it does

- Scans connected mobile storage to locate video files and related content  
- Identifies useful media folders based on file patterns and structure  
- Groups videos and subtitles into a more consistent layout  
- Prepares media for easier playback in VLC and similar desktop players  

## Scope

This repository contains the core application logic and interface for discovering, evaluating, and organizing media from mobile devices into a desktop friendly structure.
